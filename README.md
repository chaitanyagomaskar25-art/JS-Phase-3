JavaScript Projects

This repository contains beginner-level JavaScript projects built to practice DOM manipulation and user interaction.

Projects Included
 1) Quiz App

A multiple-choice quiz application that shows questions and calculates score.

2) Tip Calculator

Calculates tip amount and total bill based on user input.

 3) To-Do List

Allows users to add, complete, and delete daily tasks.

 4) Random Quote Generator

Displays random motivational quotes on button click.
